In the video “My Best Project...Until Now,” Marius Hornberger adds an automatic tool changer (ATC) to his Hammer HNC 47.82 CNC router.  

The parts he is pairing up at 15:03 are manufactured by Stepcraft:
1. The Spindle (Milling Motor)

He uses the Stepcraft MM-1000 DI milling motor (spindle).  

    Key Detail: The "DI" stands for Digitally Interfaced, meaning the CNC controller can change its speed automatically via software. It has a standard 43 mm clamping collar (Euro-neck), which is essential because the tool changer clamps directly onto this exact collar format.  
https://toolstoday.com/stepcraft-automatic-tool-changer-atc-for-hf-500-spindle.html#:~:text=The%20assembly%20is%20very%20easy:%20The%20Automatic,collet%20using%20spring%20power%20and%20pneumatically%20released.

2. The Automatic Tool Changer (ATC) Attachment

He uses the Stepcraft Automatic Tool Changer (ATC) for the MM-1000.  

    Key Detail: This is a pneumatic add-on module rather than an all-in-one ATC spindle. It slides directly over the spindle's 43 mm shaft. When Marius states he is "connecting the spindle with the changer" at 15:03, he is mechanically locking this pneumatic changer module onto the nose of the MM-1000 motor, turning a standard manual-change spindle into a pneumatic quick-release system.

3. The Tool Holders (Cones)

To make everything work, the actual router bits are mounted inside Stepcraft SK11 / ER11 tool holders. The changer module clamps onto these cones using internal spring tension and releases them using compressed air.
